# TTS-Reader-Chrome

A lightweight Chrome extension that reads selected text using the browser's native text-to-speech engine.

## Features
- Toggle speech with a hotkey (Ctrl+Shift+U).
- Right-click context menu support.
- Language auto-detection (supports Hebrew and English fallback).
- Voice selection popup.

## Installation
1. Clone this repository.
2. Open Chrome and go to chrome://extensions/.
3. Enable "Developer mode" in the top right.
4. Click "Load unpacked" and select the project folder.

## Usage
Highlight any text on a webpage and press Ctrl+Shift+U or right-click and select TTS Reader. Press the hotkey again to stop the speech.
